가장 큰 제목 태그

가장 작은 제목 태그

1. 블록 레벨 텍스트 태그


: 혼자서 한 줄을 차지하여 너비가 100% 꽉 차게 표시되는 태그
- h태그 : heading태그 - 제목을 표시하는 태그로 h1~h6까지 있다.

2) p태그 : paragraph - 단락을 표시하는 태그로 줄바꿈없이 텍스트가 한 줄로 표시됨
텍스트의 길이가 브라우저 창의 너비보다 길면 자동으로 줄이 바뀐다.

br태그 : 줄바꿈 처리는 태그로 닫는 태그가 없다

hr태그 (horizontal)
: 주제가 바뀌거나 분위기 전환을 위한 가로선 표시
닫는 태그가 없다.

div 태그(division) : 영역을 나누는 블록 태그

2. 인라인 레벨 텍스트 태그

한 줄을 차지하지 않는 태그로 화면에 표시되는 컨텐츠만큼의 너비를 차지한다.

주석 : 화면에 보이지는 않게 내부적으로 설명을 넣고자 할 때 사용하는 것
여러줄을 한꺼번에 주석처리 : ctrl+shift+/
한 줄씩 주석처리 : ctrl+shift+/
실행에서 제외하고자 하는 코드는 주석처리를 한다.
1) strong 태그 : 텍스트를 굵게 표시하는 태그

2) span 태그 : 특별한 뜻은 없지만 텍스트의 일부만 스타일 적용하고자 할 때 주로 사용

텍스트 일부만 스타일 적용 하기 굵은 텍스트

3. 목록을 만드는 태그

ul(unordered list) : 순서 없는 목록 태그

ol(ordered list) : 순서가 있는 목록 태그


ul의 list-style-type 이 기본 disc 이다

  • 메뉴1
  • 메뉴1
  • 메뉴2

ol의 list-style-type 이 기본 decimal(정수) 이다

  1. 메뉴1
  2. 메뉴2
  3. 메뉴3
숫자, 알파벳, 로마숫자로 순서를 표시한다.
시작값을 지정가능 : start

4. 표를 만드는 태그

table 태그(기본 골격), tr 태그(table row), td 태그(table data)
tabel : 행과 열의 형태로 이루어진 표의 형태를 만들 때 사용하는 태그
tr : 행에 해당하는 태그
td : 행 내부의 열에 해당하는 셀 태그
th(table head) : 행 내부의 제목에 해당하는 셀 태그

* th가 상단에 있을 경우 - 한 줄 씩 작성
